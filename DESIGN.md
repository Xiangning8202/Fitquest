## Color System · OKLCH Tokens

All neutrals carry a faint purple-blue tint (chroma 0.01–0.02). No pure `#000` or `#fff`.

```css
--void:    oklch(11% 0.012 280)   /* page background */
--surface: oklch(16% 0.015 280)   /* card backgrounds */
--edge:    oklch(24% 0.018 280)   /* borders, dividers */
--muted:   oklch(42% 0.010 280)   /* secondary text */
--subtle:  oklch(62% 0.008 280)   /* helper text */
--primary: oklch(95% 0.006 280)   /* primary text */

--brand:   oklch(60% 0.24  295)   /* core purple — mascot body, CTAs */
--neon:    oklch(72% 0.30  295)   /* neon purple — glow highlights, active icons */
--fire:    oklch(74% 0.21   48)   /* streak orange — fire, streak card only */
--success: oklch(68% 0.18  155)   /* completed states, unlocked badges */
--boss:    oklch(58% 0.26   20)   /* Boss battle red-orange */
```

**Glow rule:** Only `--brand` and `--neon` may have box-shadow glow effects. `--fire` only in streak contexts. All other colors: no glow.

## Mascot "Q" · Specification

**Species:** Abstract round creature — rabbit silhouette × bear roundness × cat expressiveness. Not explicitly any one animal.

**Color:**
- Body: `--brand` `oklch(60% 0.24 295)`
- Belly / lighter area: `oklch(70% 0.16 295)`
- Eyes: white sclera `oklch(96% 0.006 280)`, dark pupil `oklch(12% 0.015 280)`, white shine dot
- Shoes / accent: `--neon` `oklch(72% 0.30 295)`
- Cheek blush: `oklch(70% 0.18 340 / 0.35)` (subtle rose)

**Stroke:** 2.5px rounded linecaps throughout. No hard corners.

**States:**

| State | Trigger | Pose |
|-------|---------|------|
| `idle` | streak=0, empty states | Sitting, slight head tilt, eyes forward |
| `running` | WorkoutRoom active | Leaning forward, mid-stride |
| `celebrating` | Post-checkin, streak milestone | Both arms raised, squinting joyful eyes |

**Sizes:**
- `sm` 40px — inline hint
- `md` 80px — Streak Card (default)
- `lg` 120px — Streak Card featured
- `xl` 200px — empty state hero

## Streak Card · Canonical Layout

```
┌──────────────────────────────────────────────┐  ← border: --edge, left accent: 4px --brand
│  [Mascot 80px]  │  🔥 3                       │
│  celebrating    │  连续天数  (Nunito 900 52px) │
│                 │  今天也别断，保持节奏！        │
│                 │                             │
│                 │  ● ● ● ○ ○ ○ ○             │
│                 │  一 二 三 四 五 六 日          │
└──────────────────────────────────────────────┘
```

- Background: `--surface` base + `--brand` at 8% opacity gradient
- streak ≥ 7: entire card border switches to `--neon` with `box-shadow: 0 0 16px oklch(72% 0.30 295 / 0.3)`
- Mascot: `celebrating` if streak > 0 + completed today, else `idle`


Hover: glow intensity ×1.5. Tap: scale 0.97, ease-out 120ms.

---

## Anti-patterns

- No side-stripe borders (except the Streak Card brand bar, which is identity, not decoration)
- No gradient text (`background-clip: text`) — single solid color only
- No glassmorphism as default (purposeful and rare only)
- No identical 3-column icon-heading-text card grids
- No bounce or elastic in animations — ease-out-quart only
